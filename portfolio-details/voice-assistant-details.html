Voice Assistant

This project is a voice assistant built with a Raspberry Pi 3B and Google Assistant API. It was later integrated into my smart mirror project.

I followed this guide from Android Authority and this guide from the Google AIY project website in the development of this project.

Project information

  • Category: Software, Hardware
  • Date(s): September 2017 - November 2017
Components
  • Raspberry Pi 3 Model B x1
    • I used the Pi with Voice Kit image from the Google AIY project website.
  • USB Microphone x1
    • Used this simple microphone to allow the user to talk to the voice assistant.
  • Speakers x1
    • You can use a wired speaker for simple setup, or bluetooth for a slightly longer setup but clean design.
