ReadAct

Project information

  • Category: Software
  • Hackathon: DubHacks 2020
  • Date(s): October 2020
  • Source code: github.com/abhishekbabu/ReadAct
  • Partner(s): Kevin Zhu, Amartya Ranganathan, Leslie Lin

The project is a Google Chrome extension that analyzes news articles read by the user for political bias and provides additional information/links about identified political candidates and measures. We leveraged the Azure Text Analytics API for entity/key phrase extraction and queried information about the results using the WeVote API. We also leveraged the Bipartisan Press API to conduct political bias analysis on each article. The frontend of the extension was designed using HTML, CSS, Bootstrap and D3.js
