Arduino CNC Drawing Machine

This project is a 2D plotter drawing machine built using an Arduino UNO and GRBL.

Project information

  • Category: Software, Hardware
  • Date(s): July 2020 - September 2020
Components
  • STL Files
    • These are the STL files I printed in PETG to assemble the machine.
  • Arduino UNO + CNC Shield + 4 A4988 Stepper Motor Drivers + Arduino Data Cable x1
    • This kit contains all of the Arduino and CNC hardware components for this project.
    • I used 2 of the 4 stepper motor drivers in the project.
  • NEMA 17 Stepper Motor x2
    • These motors are used for the movements along the x and y axes.
  • 12V 2A Power Supply x1
    • This is used to supply power to the CNC shield and motors.
  • 12V 5A Pigtail Cable Wire x1
    • This is used to connect the power supply to the CNC shield without cutting the power supply cable to expose the inner red/black wires.
  • 3mm x 300mm Metal Round Shaft Rod Bar x4
    • These rods allow the x and y axis components to slide freely for movement during the drawing.
  • 2.7mm x 100mm Metal Round Shaft Rod Bar x2
    • These rods allow the z axis components to slide freely for movement during the drawing.
  • Servo Motor w/ Wing x1
    • This motor moves the writing utensil along the z axis to control the drawing location.
  • 6mm Timing Belt Roll + 5mm Bore Belt Pulley Wheels + 5mm Idler Pulleys x1
    • The timing belt is cut to the required length during assembly.
    • 2 of each of the bore belt pulley wheels and idler pulleys are used to translate the stepper motor rotation to linear motion for drawing.
  • 1ft x 1ft Clear Acrylic Sheet x1
    • This sheet holds the entire machine.
  • Thin Cardboard Sheet x1
    • The cardboard is cut to the size of the 3D printed base to provide a smooth drawing surface.
  • Paper Clamp x4
    • These are used to hold the paper for drawing and cardboard base on top of the 3D printed base.
  • Ultra Fine Point Sharpie x1
    • Any writing utensil will work, but I chose to use a Sharpie that I had for drawing.
  • Plastic Zipties x4
    • These are used to create loops at the ends of the timing belt strips to attach to the x and y axis components.
  • Bolts, Nuts and Spacers
    • A variable number and variable sizes of each are used to attach components together.
  • Female-Male Wires x3
    • These wires are connected to the servo motor wires to extend their length.
