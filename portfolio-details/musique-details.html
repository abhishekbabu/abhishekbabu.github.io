CoviData

Project information

  • Category: Software
  • Hackathon: DubHacks 2019
  • Date(s): October 2019
  • Source code: github.com/abhishekbabu/Musique
  • Partner(s): Aayush Chhabra,
    • Supawish Limprasert

The project is a web application that uses the pixel data from an image uploaded by the user to produce an instrumental. We leveraged OpenCV in Python for image analysis. We used the Synthesizer and Musthe libraries to map pixel data to chords and notes. The project was deployed on Google App Engine using Flask.
