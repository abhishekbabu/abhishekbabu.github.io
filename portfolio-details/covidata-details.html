CoviData

Project information

  • Category: Software
  • Hackathon: Virtual CodeDay Spring 2020
  • Date(s): May 2020 - June 2020
  • Source code: github.com/abhishekbabu/CoviData
  • Partner(s): Kevin Zhu

The project is a Google Chrome extension that displays a dashboard style view of COVID-19 country-wise stats. We leveraged Fetch API and Novel COVID API (from disease.sh) for data collection and D3.js for data visualization. We will release the extenion on the Chrome Web Store pending a few improvements and modifications.
