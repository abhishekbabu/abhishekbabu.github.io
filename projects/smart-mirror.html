Smart Mirror w/ Voice Assistant.

  • September 9, 2018
  • By Abhishek Babu

This project is a smart mirror with a voice assistant built with a Raspberry Pi and Google Assistant API. It was adapted from my previous voice assistant project.

Components

  • Raspberry Pi 3 Model B x1
    • I used the Pi with the Raspbian Stretch OS (flashed using Etcher.io).
  • Sigmac LE15AB TV x1
    • I used this small TV as the display for the mirror.
    • I removed the screen and motherboard from the TV casing and placed it in my mirror encasing.
  • Mini HDMI Cable x1
    • I used this mini HDMI cable to connect the Pi to the display inside the mirror casing.
  • 11x14 Shadowbox Frame x1
    • I used this frame as the mirror casing to hold all the components.
  • 12x18 Acrylic See-Through Mirror x1
    • I used this see-through mirror as the reflective medium that allows light to pass through from direction i.e. light from the display.
    • The mirror is reflective on the other side, allowing the users to see themselves and display at the same time.
    • I used a cutting tool (with some help) to cut this mirror to the size of the shadowbox frame.
  • USB Microphone x1
    • Used this simple microphone to allow the user to talk to the voice assistant.
  • OontZ Angle 3 Bluetooth Speaker x1
    • To make the design sleeker, I used a bluetooth speaker instead of a wired one to allow the voice assistant to talk to the user.

I followed this guide by Grensom in the development of this project.
