Raspberry Pi, Python

Reconnaissance Rover.

  • November 20, 2017
  • By Abhishek Babu

This project is a reconnaissance rover built with a Raspberry Pi and Python scripts.

Components

  • Raspberry Pi 3 Model B x1
    • I used the Pi with the default Raspbian OS.
  • Robot Car Chassis Kit x1
    • Comes with 2 DC stepper motors, 2 wheels and a 4 battery box.
  • HC-SR04 Ultrasonic Sensor x1
    • This sensor sends ultrasonic pulses and wait for an echo back when these pulses bounce off objects in front.
    • These pulses travel at the speed of sound; so if we determine the time taken for the upulse to echo back, we can use the distance formula to calculate the distance to the nearest object in front.
  • L298N Motor Controller x1
    • The controller is used to manage the power output to each motor, which allows for forward movement, backward movement and turning.
  • IR Sensor x2
    • I mounted these sensors on the back in case there were obstacles found while reversing.
    • These sensors have a built-in range up to which they detect obstacles, so there's no process to determining distances; the sensors send a signal if they encounter an object within their effective range.
  • Solderless Breadboard x1 400 Tie and x2 170 Tie
    • I used the 400 tie breadboard to mount the ultrasonic sensor and temperature-humidity sensor.
    • I used the two 170 tie breadboards to mount the IR sensors on the back of the rover.
  • Portable USB Charger x1
    • The charger was used as the power source for the Pi.
  • Metal Model Building Set x1
    • I used this set to build a frame atop the chassis where the Pi would be mounted.
    • I used the frame to support the breadboards required for the ultrasonic and infrared sensors.
  • Amadget 5MP Raspberry Pi Camera Module x1
    • Here, you can find information about how to livestream using a Pi camera.
  • DHT22/AM2302 Digital Temperature and Humidity Sensor x1
    • Here, you can find information about how to set up the Python scripts to run the temperature-humidity sensor.
