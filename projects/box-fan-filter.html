Box Fan Filter Attachment.

  • April 26, 2020
  • By Abhishek Babu

This project involved remixing and 3D printing a box fan filter attachment that would allow me to attach mutliple air filters to a box fan, allowing it to act as an air purifier.

I used TinkerCAD to remix the model for a single filter into one that could support two filter. I sliced the model using Ultimaker Cura and 3D printed it using my Creality CR-10 3D printer.
